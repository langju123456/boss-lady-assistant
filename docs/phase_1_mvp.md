# Phase 1 MVP

## Goal

Help one real salon owner finish daily business reporting and receive useful advice without using Excel.

## Main Flow

```text
Voice/Text Input
→ Business Recorder Agent
→ SQLite
→ Business Analyst Agent
→ Business Advisor Agent
→ Plain Chinese answer
```

## What Is Included

- Natural language business recording
- SQLite storage
- Daily, weekly, monthly reports
- Marketing content generation
- Business advice based on recent data

## What Is Not Included

- Authentication
- Multi-store management
- Payments
- Complex CRM
- WeChat official integration
- Staff permissions

## Next Iteration

1. Add Whisper voice input.
2. Add OpenAI structured extraction for messier speech.
3. Add WeChat-friendly share/export.
4. Add a seven-day onboarding script for the salon owner.
