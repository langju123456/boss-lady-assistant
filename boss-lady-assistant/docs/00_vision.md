# Vision

## Why This Project Exists

Small beauty salon owners often do not use Excel, CRM, or digital systems. They rely on memory, paper notes, WeChat chats, and personal experience.

The goal of 老板娘经营助手 is to help them run their business through simple voice or text interaction. The owner should be able to say what happened in the shop, ask what is going wrong, request activity ideas, and receive useful business output without learning new software.

## Target Users

* Small county-level beauty salon owners
* 1-10 employees
* Limited digital skills
* Prefer WeChat and voice communication
* Need help with daily recording, business review, customer follow-up, and marketing content

## Core Hypothesis

The user does not want to learn software.

The user wants to talk naturally and receive useful business output.

## Product Vision

One chat interface.

Four backend agents:

1. Business Recorder Agent
2. Business Analyst Agent
3. Content Operator Agent
4. Business Advisor Agent

## Current Phase

MVP phase.

Focus on one real salon first.
